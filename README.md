# dinarys restapi
Test task of API implementation

 ### Installation
  `git clone https://github.com/Scanerrr/dinarys_rest.git`

 
  `composer install`

  `npm install`

 `Set up DB_* params in .env-file`
 
 `php artisan migrate`
 
 `php artisan db:seed`

 ### Getting started
  `php artisan serve`

 ### Running tests
 `composer test`
